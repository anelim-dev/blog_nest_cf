# Development environment

1. Clone the repository
2. Execute:
```
yarn install
```
3. Install Nest CLI 

```
npm i -g @nestjs/cli
```

4. Upload the database
```
docker-compose up -d
```

5. Execute the app in dev
```
yarn start:dev
```

## Stack
* MongoDB
* Nest
